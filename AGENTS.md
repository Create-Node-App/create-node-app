# Agent Instructions for create-node-app

## Overview

`create-node-app` is a CLI tool for scaffolding Node.js projects from templates.

## Tech Stack

- TypeScript
- Node.js 22+
- npm 10+
- Turborepo (monorepo)

## Key Commands

```bash
# Build all packages
npm run build

# Run tests
npm run test:all

# Lint
npm run lint

# Type-check
npm run type-check
```

## Code Style

- ESLint flat config
- Prettier for formatting
- TypeScript strict mode
